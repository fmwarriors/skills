## Notes

- All documentation is consistent across versions from FileMaker Pro 14 onwards
- The specification is a subset of the full W3C SVG standard
- Only UTF-8 encoding is supported (UTF-16 not supported)
- User coordinate units only (no unit symbols)
- Documentation assumes basic knowledge of computer graphics and SVG concepts
